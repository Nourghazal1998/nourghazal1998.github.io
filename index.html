Crack Width Measurement POC

Built a system that measures crack width in mm from krkCMd dataset images. YOLOv11 with tiled inference handles the segmentation, then width is measured along the crack skeleton using the image GSD. Includes a web app with live inference, per-crack stats, and PDF report generation.

7.7% mean error 91% match rate 6 images validated
